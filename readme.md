
# Huerkle-Puzzo (Hupu)

Команда на генерацию 40000 головоломок с длиной пароля (пароль порождает ключ алгоритма симметричного шифрования) 2 байта:
```sh
cargo run -- --mode generator --puzzle-amount 40000 --password-length 2 >puzzles.txt
```

Все созданные головоломки передаются пользователю.


Пользователь самостоятельно выбирает головоломку, которую хочет попытаться решить. Свой выбор он держит в секрете.
Пусть пользователь решил взломать головоломку, `79d4cef514852aa49db6910dfb84e4a49c4dc046a2ea8bd0` (строка байтов).

Информация о головоломке у генерирующей стороны:
```
Puzzle no: 3862237000
Key no: 696006766
Puzzle: 79d4cef514852aa49db6910dfb84e4a49c4dc046a2ea8bd0
Key: 302fa4e916e11f2cd215d39ddaebd58e182cf8885406fcddfc087bbdaac2282d
Password: b4c4
```

Команда на поиск решения головоломки:
```sh
cargo run -- --mode solver --password-length 2
```

Пользователь указывает головоломку, которую хочет решить (взломать):
```
Puzzle: 79d4cef514852aa49db6910dfb84e4a49c4dc046a2ea8bd0
```

Если головоломка будет успешно решена, программа выведет сообщение об успешном решении:
```
Solved
Puzzle number: 3862237000
Secret key number: 696006766
Key: 302fa4e916e11f2cd215d39ddaebd58e182cf8885406fcddfc087bbdaac2282d
```
